About Site Builder

Site Builder automates the creation of new websites. Give it a project name and a short description — it takes care of everything else and hands you back a live URL.

What happens when you submit

  1. A new GitHub repository is created from a standard template.
  2. A Cloudflare Pages project is created and linked to that repository.
  3. Claude Code reads your description and writes the initial site content, then commits and pushes it.
  4. Cloudflare Pages detects the push and deploys the site automatically.

Within a couple of minutes your site is live at {project-name}.pages.dev.

What you get

A fully deployed static website with AI-generated content based on your description, version-controlled on GitHub, and hosted on Cloudflare Pages — ready for you to continue building with Claude Code.

What you need to provide

  • A project name (used as the repo name and the subdomain)
  • A paragraph describing what the site is for

That's it.
